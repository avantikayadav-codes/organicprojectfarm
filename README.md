# 🌿 Organic Farm - Java Web Application

A dynamic organic products web application built with Java Servlets and JSP, 
allowing users to explore organic products, register, login, and manage their cart.

---

## ✨ Features

- 🔐 **User Registration & Login** — Secure authentication system
- 🛒 **Shopping Cart** — Add and manage products
- 🖼️ **Image Upload** — Product image management
- 🏠 **Home Page** — Beautiful landing page
- 🌿 **Products Display** — Browse organic products
- 📝 **Blog Section** — Organic farming articles
- 👥 **Team Section** — Meet the team
- ℹ️ **About Us** — Farm story
- 🌟 **Testimonials** — Customer reviews
- 📞 **Contact Page** — Get in touch

## ⚙️ How to Run

1. Clone this repository
2. Import in **Eclipse IDE** as Dynamic Web Project
3. Configure **MySQL** database
4. Update DB credentials in `DBconfig.java`
5. Deploy on **Apache Tomcat** server
6. Open browser → `http://localhost:8080/organicprojectfarm`
